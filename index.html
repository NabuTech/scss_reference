Styled Box with Variables

Understanding SCSS Variables

SCSS allows us to use variables to store and reuse values throughout our stylesheets. This promotes consistency, makes it easier to update styles globally, and enhances maintainability.

What We've Done in This Example

In our SCSS file (_variables.scss), we have defined variables for colors, fonts, sizes, and borders. These variables are then used to style various elements in our HTML, such as the header, content section, and a styled box.

Variable Examples

  • Colors: $primary-color, $secondary-color, $background-color
  • Fonts: $main-font
  • Sizes: $base-font-size, $heading-font-size
  • Borders: $box-border-radius, $box-border

Feel free to experiment with these variables and observe how changes in the variables impact the overall styling of the page.
